Tipos de imputs

Estos solo son algunos tipos de imputs

  • button: Crea un botón que se puede hacer clic.
  • checkbox: Crea una casilla de verificación que se puede marcar o desmarcar.
  • color: Crea un control para seleccionar un color.
  • date: Crea un control para seleccionar una fecha.
  • datetime-local: Crea un control para seleccionar una fecha y hora.
  • email: Crea un control para ingresar una dirección de correo electrónico.
  • file: Crea un control para seleccionar un archivo
  • hidden: Crea un control oculto que no se muestra en la página.
  • image: Crea un control para mostrar una imagen.
  • month: Crea un control para seleccionar un mes y un año.
  • number: Crea un control para ingresar un número.
  • password: Crea un control para ingresar una contraseña.
  • radio: Crea un botón de opción que se puede seleccionar.
  • range: Crea un control para seleccionar un rango de valores.
  • reset: Crea un botón para restablecer los valores de un formulario.
  • search: Crea un control para ingresar texto de búsqueda.
  • submit: Crea un botón para enviar un formulario.
  • tel: Crea un control para ingresar un número de teléfono.
  • text: Crea un control para ingresar texto.
  • time: Crea un control para seleccionar una hora.
  • url: Crea un control para ingresar una dirección URL
  • week: Crea un control para seleccionar una semana.
